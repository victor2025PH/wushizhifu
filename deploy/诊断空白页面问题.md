# 诊断空白页面问题

## 问题分析

前端成功构建，Nginx 返回 200，但页面显示空白。

## 可能原因

1. **`index.html` 缺少 JavaScript 引用**
   - Vite 构建时应该在 `index.html` 中自动注入 `<script type="module" src="/assets/index-xxxxx.js"></script>`
   - 如果缺少这个引用，React 应用不会加载

2. **JavaScript 文件路径错误**
   - 构建的文件可能使用了绝对路径，但在服务器上需要相对路径

3. **构建输出不完整**
   - `dist/assets/` 目录可能不存在或为空

## 检查步骤

在服务器上执行：

```bash
# 1. 检查构建输出
cd /home/ubuntu/wushizhifu/frontend/dist
ls -la

# 2. 检查是否有 assets 目录
ls -la assets/

# 3. 检查 index.html 内容
cat index.html | grep -i script

# 4. 查看完整的 index.html
cat index.html
```

## 解决方案

如果 `index.html` 中缺少 JavaScript 引用，需要：

1. 确保 Vite 正确构建
2. 检查 `vite.config.ts` 配置
3. 可能需要重新构建

## 快速修复

```bash
cd /home/ubuntu/wushizhifu/frontend

# 清理并重新构建
sudo chown -R ubuntu:ubuntu dist
rm -rf dist/*
npm run build

# 检查构建结果
ls -la dist/
cat dist/index.html | grep script

# 设置权限
sudo chown -R www-data:www-data dist
sudo chmod -R 755 dist
sudo systemctl reload nginx
```
