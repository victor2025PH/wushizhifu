# 支付指南功能說明

## 功能概述

當用戶點擊"支付寶"支付按鈕時，會顯示一個支付操作指南彈窗，包含：
1. 詳細的文字操作說明（5個步驟）
2. 觀看視頻教程按鈕
3. 視頻播放器（支持播放、暫停、靜音、全屏）
4. 關閉按鈕
5. 確認按鈕（繼續支付）

## 新增文件

1. **components/AlipayGuideModal.tsx** - 支付指南彈窗組件
2. **components/VideoPlayer.tsx** - 視頻播放器組件
3. **public/video_20051225.mp4** - 支付操作視頻文件

## 修改文件

1. **components/Dashboard.tsx** - 添加彈窗狀態和處理邏輯
2. **types.ts** - 添加中英文翻譯文本

## 功能特點

- ✅ 響應式設計，適配移動端
- ✅ 動畫過渡效果
- ✅ 視頻播放控制（播放/暫停、靜音、全屏）
- ✅ 中英文雙語支持
- ✅ 清晰的步驟說明
- ✅ 溫馨提示信息

## 部署步驟

### 1. 確保視頻文件在 public 目錄

視頻文件應位於：`public/video_20051225.mp4`

### 2. 推送代碼到 GitHub

```bash
cd wushizhifu-full

git add .
git commit -m "Add Alipay payment guide modal with video tutorial"
git push
```

### 3. 在服務器上更新

```bash
cd ~/wushizhifu/frontend

# 拉取最新代碼
git pull

# 確保視頻文件在 public 目錄
ls -lh public/video_20051225.mp4

# 構建前端
sudo chown -R ubuntu:ubuntu dist
npm run build

# 檢查構建結果（確認視頻文件被複製到 dist）
ls -lh dist/video_20051225.mp4

# 恢復權限
sudo chown -R www-data:www-data dist
sudo chmod -R 755 dist
sudo systemctl reload nginx
```

## 使用流程

1. 用戶點擊 Dashboard 上的"支付寶"卡片
2. 顯示支付指南彈窗（包含文字說明和"觀看視頻教程"按鈕）
3. 用戶可以：
   - 閱讀文字說明
   - 點擊"觀看視頻教程"按鈕播放視頻
   - 在視頻播放器中控制播放
   - 點擊"我知道了，繼續支付"進入支付流程
   - 點擊"取消"關閉彈窗

## 視頻文件部署

視頻文件需要部署到服務器的 `public/` 目錄，Vite 會在構建時自動複製到 `dist/` 目錄。

如果需要在服務器上手動上傳視頻文件：

```bash
# 使用 WinSCP 上傳到：
# /home/ubuntu/wushizhifu/frontend/public/video_20051225.mp4

# 然後重新構建
cd ~/wushizhifu/frontend
sudo chown -R ubuntu:ubuntu dist public
npm run build
sudo chown -R www-data:www-data dist
sudo chmod -R 755 dist
sudo systemctl reload nginx
```
